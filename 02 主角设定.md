## 花如意

| 基础信息      | 内容                                         |
| --------- | ------------------------------------------ |
| **性别**    | 女                                          |
| **年龄**    | 18 岁                                       |
| **职业**    | 花隐山庄庄主（继承中）·花家守山人传承者（尚不知晓）                 |
| **服装关键色** | 浅杏色为底、白色为辅，袖口领口绣小碎花；常年有一处沾着泥或草渍，永远是刚干完活的样子 |
| **随行**    | 粉色小鸟[[霓裳]]，几乎不离肩                           |

**性格：**  
好奇心旺、行动先于思考、嘴硬心软。清醒但冒失——知道自己要做什么，过程总出岔子。不爱认输，但会认错。对熟悉的人话多，对陌生人沉得住气。

**口癖：**
- "我来弄！"（然后帮倒忙）
- "没事，我会的。"（然后发现不会）
- "……好吧，你说得对。"（认输，但语气很快）

**成长弧光：**
```
归来的困惑少女
  → 扛起庄子的小庄主
    → 第一次理解"娘为什么要走"（守山人的真正含义）
      → 不再是寻母的孩子，是花隐山庄的花如意，也是下一代守山人
```

## 花芳华（缺席的母亲）

| 基础信息 | 内容 |
|---------|------|
| **年龄** | 约 42 岁 |
| **职业** | 医术精湛的女大夫，游走江湖，毕生研究山中奇异草木与花卉 |
| **家族身份** | 花家守山人，是这一代传承者 |
| **失踪原因（表层）** | 在花隐山中发现从未记载的奇异花卉，一路追查记录，越走越远 |
| **失踪原因（真实）** | 五年前在花隐山阻止幽篁夫人夺取山神（福宝），将福宝带回山庄安置后，为避免幽篁夫人追踪到庄子，主动离开、拉开距离，以游走的方式引开视线 |

**人物特质：** 学术型"野外研究狂人"——研究山中奇异草木是守山人职责，也是真心喜爱。温柔却有铁骨，护住该护的东西时从不手软，只是从来不解释。

**守山人的传承方式：**

> 花家的规矩：守山人不说，等下一代自己看见。  
> 花芳华当年是如此，她娘也是。

**她留给如意的东西：**

| 遗留物 | 位置 | 解锁时机 |
|--------|------|---------|
| 《百花药录》手记（残缺） | 阁楼藏书室 | 第7章 |
| 各院落的花卉（隐藏线索） | 花隐山庄各处 | 随修缮进度 |
| 托石伯转交的小物件 | 石伯的记忆里 | 每次院落修缮后 |
| 后山结绳记号 | 后山秘径 | 第9章 |
| 竹签"等你" | 山腰平台（后山花圃处） | 第9章 |
| 守山人传承手书（残页） | 藏书室夹层 | 第11章 |

**她托人带回的口信风格：**
> *"一切安好，发现新物种，附上图样，请帮我查对一下是否有记载。"*  
> （没有一封提到"我什么时候回来"）

**真相：** 她从未出事，一直在外兜圈子、引走幽篁夫人的注意，同时继续追寻山中奇异花卉，写满了三本手记——等庄子修好，等如意长大，等那个邪修放弃，她就回来。→ 详见[[12 幕后线索彩蛋]]
