## 花如意

| 基础信息 | 内容 |
|---------|------|
| **性别** | 女 |
| **年龄** | 18 岁 |
| **职业** | 花隐山庄庄主（继承中） |
| **服装关键色** | 浅杏色为底、白色为辅，袖口领口绣小碎花；常年有一处沾着泥或草渍，永远是刚干完活的样子 |
| **随行** | 粉色小鸟[[霓裳]]，几乎不离肩 |

**性格：**  
好奇心旺、行动先于思考、嘴硬心软。清醒但冒失——知道自己要做什么，过程总出岔子。不爱认输，但会认错。对熟悉的人话多，对陌生人沉得住气。

**口癖：**
- "我来弄！"（然后帮倒忙）
- "没事，我会的。"（然后发现不会）
- "……好吧，你说得对。"（认输，但语气很快）

**成长弧光：**
```
归来的困惑少女
  → 扛起庄子的小庄主
    → 第一次理解"娘为什么要走"
      → 不再是寻母的孩子，是花隐山庄的花如意
```

## 花芳华（缺席的母亲）

| 基础信息 | 内容 |
|---------|------|
| **年龄** | 约 42 岁 |
| **职业** | 医术精湛的女大夫，游走江湖，毕生研究"幽兰草"药性 |
| **失踪原因** | 发现异常节气开花的幽兰草"花母"，一路追着花期研究，越走越远，根本没意识到家里急坏了 |

**人物特质：** 学术型"野外研究狂人"——性格温柔却有铁骨，对幽兰草的热情远大于对日常生活的关注。

**她留给如意的东西：**

| 遗留物 | 位置 | 解锁时机 |
|--------|------|---------|
| 《百花药录》手记（残缺） | 阁楼藏书室 | 第7章 |
| 各院落的花卉（隐藏线索） | 花隐山庄各处 | 随修缮进度 |
| 托石伯转交的小物件 | 石伯的记忆里 | 每次院落修缮后 |
| 后山结绳记号 | 后山秘径 | 第9章 |
| 竹签"等你" | 山腰平台 | 第9章 |

**她托人带回的口信风格：**
> *"一切安好，发现新物种，附上图样，请帮我查对一下是否有记载。"*  
> （没有一封提到"我什么时候回来"）

**真相：** 她从未出事，只是追花追得太投入，一路走了三个省，等她回过神来已不知身在何处。→ 详见[[12 幕后线索彩蛋]]
