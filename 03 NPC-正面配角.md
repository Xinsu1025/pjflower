## 石来喜（石伯）— "世界上最慢的快递"

| 基础信息      | 内容                                    |
| --------- | ------------------------------------- |
| **性别**    | 男                                     |
| **年龄**    | 约 65 岁                                |
| **职业**    | 花隐山庄老管家，在花家服侍了一辈子                     |
| **服装关键色** | 深灰色旧棉袍，领口袖口浅蓝滚边，总穿得整整齐齐；腰间别着一串钥匙，走路轻响 |
| **喜好物件**  | [[07 喜好物件系统#石来喜·老姜红糖茶\|老姜红糖茶]]        |

**性格：**  
老实、忠厚、礼数严，没什么心眼，一辈子认准了一件事就做到底。记性越来越差，越重要的东西越容易忘——等到某个契机被触发，才拍脑袋想起来。对花如意藏着极深的疼爱，但表达方式是热粥和唠叨。

**口癖：**
- "夫人说……"（说着说着停下来，想一会儿，再接着说）
- "老奴这就去。"
- "小小姐，这是……夫人让老奴转交的。老奴……忘了。"

**健忘触发机制（玩法钩子）：**  
玩家完成一处院落修缮 → 石伯被触发，颤颤巍巍捧出芳华托付的物件 → 解锁剧情记忆或道具奖励。

> 详见[[12 幕后线索彩蛋]] — "石伯忘事的边界"

**成长弧光：**
```
独自守庄六年，只有福宝和一座空庄子
  → 花如意归来，像压了多年的石头终于可以放下
    → 庄子一处处修好，记忆一件件被唤醒，使命一点点交还
      → 终章：把最后一件东西递出去，说"老奴这次没忘"
```
核心转变：**放下**——守了一辈子，终于可以不用再一个人扛着了。

## 福宝 — "后山第一探险家"

| 基础信息      | 内容                                                  |
| --------- | --------------------------------------------------- |
| **性别**    | 未知                                                  |
| **年龄**    | 外表看约 5 岁                                            |
| **真实身份**  | 花隐山山神，在此山已驻守数百年                                     |
| **化身**    | 千年人参（天地灵物），五年前险被幽篁夫人所夺，由花芳华出手救下后以孩童形态寄居山庄         |
| **来历**    | 五年前花芳华将其带回山庄托石伯看护，对外以"孤儿"示人                         |
| **服装关键色** | 嫩草绿小袄，下摆绣歪歪扭扭的小花（她自己绣的），头上两个小揪揪，常有一边散掉            |
| **喜好物件**  | [[07 喜好物件系统#福宝·五色糖人\|五色糖人]]                         |

**性格：**  
外表古灵精怪，像个寻常孩童，但说出的话时常让大人发愣。对山里的一切了如指掌，能与山中精怪自然交流——因为那些精怪本就是她守护之地的一部分。偶尔说出超出五岁孩子认知的话，自己浑然不觉，说完又去追蝴蝶了。

> ⚠️ **核心秘密：** 福宝是花隐山山神的化身，花隐山"四季同框、百花不灭"的奇景，正是因为山神驻守此地。若福宝离开或消失，这片山的灵气将随之散尽，百花凋零。花家世代为守山人，花芳华当年是为了护住山神，才将福宝带回并藏于山庄。→ 详见 [[12 幕后线索彩蛋]]

**口癖：**
- "我知道！"（不一定真的知道，但非常确定地说）
- "后山那边有——"（十句话里六句开头是这个）
- "娘说……"（说到一半有时停，但很快接着说完，不沉浸在难过里）
- "这里我认识的。"（进入任何山中区域，都这么说，理所当然）

**后山精怪设定：**

| 精怪 | 外形 | 性格 | 与福宝的关系 |
|------|------|------|------------|
| **苔小团** | 长得像一团会动的青苔，圆滚滚 | 认生，一有动静就缩成球 | 山神护持下生出灵性的小精，福宝一叫它就现身 |
| **花灯儿** | 像萤火虫但是白色，在花丛中飞 | 喜欢跟着人，不喜欢被抓 | 引路用的——福宝在后山从不迷路，全靠花灯儿 |
| **石爷爷** | 一块常年坐在溪边的老石头，偶尔睁眼 | 古板话少，爱发呆 | 守山的老灵，见过花家几代守山人，偶尔认得出花芳华的气息 |

**神力边界设定（20%神奇元素）：**

| 能力 | 表现形式 | 玩家发现时机 |
|------|---------|------------|
| 感知山中气息 | 能嗅辨来人踪迹，脚印、气息全无遮掩 | 第5章脚印场景 |
| 与精怪沟通 | 精怪视她为主，自然听令 | 全程穿插 |
| 山庄花卉护持 | 她在的地方，花不会凋谢 | 第11章后山花圃盛开夜 |
| 偶发预感 | 说出某事"要发生了"，往往成真 | 零散剧情节点 |

**成长弧光：**
```
以孩童形态藏于山庄，只有石伯一人看护
  → 有了如意姐姐，庄子有了人气，山也慢慢"醒"了
    → 从藏着自己到主动带大家探索后山——她守护这里，也信任这里的人
      → 终章：娘回来了，幽篁夫人离去，她终于不用藏着——在花圃里睡着，周围百花齐开
```
核心转变：从**藏匿**到**被接纳**。

## 周朗 — "有担当的帅大哥"

| 基础信息 | 内容 |
|---------|------|
| **性别** | 男 |
| **年龄** | 20 岁 |
| **职业** | 仙湖镇木匠，家传手艺，镇上公认手最巧 |
| **服装关键色** | 靛蓝色短褂，肩宽，袖子永远是挽起来的；身上有松木香和淡淡汗味，工具袋随身，腰间别着刨子 |
| **喜好物件** | [[07 喜好物件系统#周朗·桂花酥\|桂花酥]] |

**性格：**  
开朗正直，说话直接，天然有担当感——不用开口，他已经站在最需要有人站的地方了。对花如意有感情，既不掩藏也不急于表白，踏踏实实地等着合适的时候说。

**口癖：**
- "交给我。"
- "放心。"
- "我去看看。"

**代表场景：**

> 花如意："周朗，我来帮你抬这根梁——"  
> 周朗：（把她拦住）"不用。"  
> 花如意："你一个人怎么抬？"  
> 周朗：（指了指旁边的滑轮装置）"……我做了工具。"  
> 花如意："那我去帮你递钉子？"  
> 周朗：（把钉子袋挂到自己腰上）"不用。"

**成长弧光：**
```
花如意不在的六年，每年来庄子帮石伯修葺，没人问他为什么
  → 花如意回来，做了六年的事终于有了意义
    → 从"帮如意扛下所有事"到"学会相信她也可以"
      → 终章：花如意问"不委屈吗"，他说"没有"，顿一顿，"值得等"
```

## 苏凝 — "安静的知己"

| 基础信息 | 内容 |
|---------|------|
| **性别** | 女 |
| **年龄** | 18 岁 |
| **职业** | 苏氏绣坊千金，家里是镇上最大的绣坊 |
| **服装关键色** | 月白色或浅竹青色为主，衣料素净；发髻梳得干净，只用一根素木簪 |
| **喜好物件** | [[07 喜好物件系统#苏凝·双股碧玉绣线\|双股碧玉绣线]] |

**性格：**  
话少，但观察力极强，说出来的每句话都很准。情报自然汇集到她那里（大家都愿意找她说话）。与花如意是一动一静，她的存在让花如意的热闹有了一个可以落回来的地方。

**口癖：**
- "……嗯。"
- "你看这个。"
- （偶尔说出一句一针见血的话，然后若无其事地继续绣）

**成长弧光：**
```
安静地陪在花如意身边，几乎不发声
  → 发现自己不动声色收集的消息成了关键线索
    → 第一次主动开口，说出一件她观察了很久却从未说过的事
      → 终章：花如意说"你其实什么都懂"，苏凝低头继续绣，嘴角动了一下
```

## 裴昀 — "同龄最帅，带着秘密的来客"

| 基础信息      | 内容                                     |
| --------- | -------------------------------------- |
| **性别**    | 男                                      |
| **年龄**    | 19 岁                                   |
| **职业**    | 来历不明的游历少年，以"借宿客人"身份住进山庄                |
| **服装关键色** | 雪白色外衫，内衬浅灰，腰间一块碧色暖玉；在庄子住久了之后，衣领会慢慢松开一点 |
| **喜好物件**  | [[07 喜好物件系统#裴昀·一壶无名山泉\|一壶无名山泉]]        |

**性格：**  
同龄人里气质最出挑，好看得让人想多看一眼，但他自己浑然不觉。知道[[02 主角设定#花芳华（缺席的母亲）|花芳华]]失踪的部分真相，有自己的顾虑，不急于说出来。被庄子的热闹一天天"化"着。

**口癖：**
- "有意思。"（嘴角有一点弧度）
- "……还好。"
- （被福宝盘问感情问题时沉默两秒，然后说）"你几岁？"

**与花芳华的关系：** 他是芳华研究途中结识的晚辈，受托照看花如意。→ 详见 [[12 幕后线索彩蛋]]

**成长弧光：**
```
带着秘密和使命来到庄子的少年
  → 以为住几天就会离开，却一天天留了下来
    → 庄子的热闹填进了他身上某处空着的地方
      → 终章：主动说出他知道的一切——不是因为使命，是因为他选择了这里
```

## 冯叙 — "茶馆掌柜，压了二十年感情的男人"

| 基础信息 | 内容 |
|---------|------|
| **性别** | 男 |
| **年龄** | 38 岁 |
| **职业** | 仙湖镇"听雨阁"茶馆掌柜，镇上最有阅历的男性商人 |
| **服装关键色** | 灰蓝色长衫，腰间常系一条布围裙，袖口有茶渍洗不净；头发束得随意，有几缕散落 |
| **喜好物件** | [[07 喜好物件系统#冯叙·茉莉鲜花茶\|茉莉鲜花茶]] |

**性格：**  
处事圆滑，说话留余地。年轻时暗恋过[[02 主角设定#花芳华（缺席的母亲）|花芳华]]，从未说出口。看见花如意，像看见芳华年轻时，心里又高兴又难受。给花如意的建议大多是对的，但自己的事一团糟。

**口癖：**
- "你娘啊……"（说到一半，端起茶碗）
- "当年……"（又说了半句，又停了）
- "进来喝杯茶。"（他表达关心的唯一方式）

**成长弧光：**
```
把对芳华的感情压了二十年，化成茶叶沉在壶底
  → 花如意回来，他终于有了可以做点什么的机会
    → 把那份藏了很久的感情，转换成帮这个孩子找到她娘的行动
      → 终章：花如意母女重逢，他站在茶馆门口，转身进去，把那壶茶彻底倒了，重泡一壶新的
```
