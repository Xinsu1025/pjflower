## 核心美学定位

> **山中日常，百花常开，明初田园，笔墨轻盈。**

本游戏以**工笔淡彩**为基础视觉语言，追求一种"不浓艳，不晦暗，刚刚好"的明代民间日常质感。

---

## 视觉风格

| 维度 | 方向 | 参考说明 |
|---|---|---|
| 整体画风 | 工笔淡彩×水墨晕染混合 | 主体用线条勾勒，背景用淡墨渲染 |
| 角色风格 | 简洁明代民间服饰，不繁复 | 避免戏服化，追求"普通人"质感 |
| 场景风格 | 院落为主，山色为辅 | 重点刻画花卉、木构、青石板细节 |
| 光影处理 | 柔光为主，无强烈阴影 | 制造"温暖下午"的恒定感 |
| 花卉表现 | 精细描绘，各章节有标志性花卉 | 幽兰、茉莉、石榴花、竹为主要植物符号 |

## 主色调系统

| 色彩名 | 色调描述 | 使用场合 |
|---|---|---|
| 杏白 | 温暖米白，带一点橙意 | 主界面底色、白天场景基调 |
| 竹青 | 清透绿灰，不饱和 | 植物、户外场景、自然元素 |
| 幽紫 | 淡紫微蓝，若隐若现 | 山神相关场景专属色，重要剧情节点高亮 |
| 枯草褐 | 沉稳黄棕，有岁月感 | 木构建筑、旧物件、石伯相关场景 |
| 点睛朱 | 低饱和红，不刺眼 | UI重要按钮、节日/喜庆小点缀 |

> **禁用色系：** 高饱和荧光色、深重黑色大面积使用（仅幽篁夫人相关场景可用深色）

## UI 风格

| 元素 | 设计方向 |
|---|---|
| 整体 UI 框架 | 仿宣纸卷轴边框，轻木质感按钮 |
| 字体选择 | 衬线宋体（正文）+ 手写风格（日记/信件内容） |
| 按钮反馈 | 轻柔音效 + 墨迹扩散小动画 |
| 对话框 | 仿信纸格式，边角有轻微折痕纹理 |
| 物件图鉴 | 手绘卡片风格，附简短手写说明 |
| 地图/院落全图 | 鸟瞰工笔图，修缮后区域用饱和色填充 |

## 音乐基调

| 场景类型 | 音乐方向 | 乐器偏好 |
|---|---|---|
| 主界面 | 轻柔开场，有仙湖镇的"远山感" | 古筝主旋律 + 笛声点缀 |
| 寻物场景进行中 | 轻快专注，不焦虑，有节拍感 | 琵琶 + 木鱼 + 轻鼓 |
| NPC 互动对话 | 随角色而变（见下表） | — |
| 剧情推进节点 | 情绪拉满，但不悲剧 | 胡琴 + 古筝 + 钟磬 |
| 幽篁夫人相关 | 轻微神秘感，竹林风声 | 洞箫 + 竹声音效 |
| 终章·母亲归来 | 无歌词轻哼旋律，温暖 | 人声哼唱 + 古筝 |

### NPC 专属音乐色彩

| NPC | 音乐色彩 | 代表乐器 |
|---|---|---|
| 石伯 | 沉稳、年代感、有余韵 | 低音胡琴 |
| 福宝 | 轻快跳跃、有一点野外感 | 短笛 + 蟋蟀虫鸣 |
| 周朗 | 简洁平静、干净 | 古琴拨弦，无装饰音 |
| 苏凝 | 细腻、安静、有绣针般的精准感 | 扬琴轻音 |
| 裴昀 | 淡然、略有距离感 | 箫声，偶有回响 |
| 冯叙 | 圆润、世故中带温情 | 琵琶中段，流畅 |
| 金不换 | 浮夸热闹，但有喜感 | 唢呐 + 轻喜剧鼓点 |
| 幽篁夫人 | 深邃、克制、最后转为平和 | 低音洞箫 |

## 音效设计

| 场景 | 音效描述 |
|---|---|
| 寻物点击成功 | 轻柔的"发现"叮响声 + 花瓣飞散粒子 |
| 区域修缮完成 | 舒缓的"完成"钟声，轻而远 |
| NPC 好感提升 | 花瓣飘落音效（不同NPC不同花） |
| 后山花圃开花动画 | 极轻的"绽开"声 + 风声 |
| 石伯说完话 | 背景偶有远处鸟鸣 |
| 金不换出场 | 折扇打开的"啪"声 |
| 霓裳飞过 | 羽翼轻拍声，玫粉色粒子 |

## 季节与天气视觉

| 时段 | 视觉处理 |
|---|---|
| 晴天 | 明亮杏白光，花瓣偶有飘落 |
| 雨天 | 竹青调加深，雨声系统音效，水波纹UI元素 |
| 黄昏 | 幽兰紫染天边，柔光打亮院落 |
| 节气特殊日 | 对应花卉全景开放动画（如茉莉季、幽兰季） |
