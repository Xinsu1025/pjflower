## 核心机制来源：浪漫餐厅（Delicious）

本游戏以**浪漫餐厅系列**的关卡合并机制为核心原型，结合院落修缮与剧情解锁，将"时间管理"重心转移至"空间修缮+情感互动"。

---

## 机制映射表

| 浪漫餐厅原机制 | 花开寻踪的对应设计 | 目的 |
|---|---|---|
| 关卡内合并食材/订单 | 合并修缮材料（木料+木料=梁木） | 给玩家动手感，短循环正反馈 |
| 升级厨房/餐厅设施 | 修缮院落各功能区 | 空间进度可视化 |
| VIP客人特殊需求 | NPC特定喜好物件触发特殊对话 | 角色情感加深 |
| 关卡限时星级评分 | 每章修缮目标达成度（非强制限时） | 降低焦虑感，保持探索氛围 |
| 故事推进解锁新关卡 | 修缮进度解锁剧情章节 | 修与看合一 |

---

## 关卡解锁节奏

```
第一幕（立足篇）
  合并关卡：小型修缮，院路、花圃外围
  → 解锁：茶室、正院
  → 剧情解锁：序章 + 第1-3章

第二幕（麻烦篇）
  合并关卡：中型修缮，厨房、水榭、阁楼
  → 解锁：后院矮墙、阁楼藏书室
  → 剧情解锁：第4-8章

第三幕（真相篇）
  合并关卡：大型修缮，后山路、幽兰草圃
  → 解锁：山腰平台、全景山庄
  → 剧情解锁：第9-12章
```

## NPC互动节奏设计

每位 NPC 有三层互动进度：

1. **出场层**：NPC 初次登场，基础对话解锁  
2. **喜好层**：赠送对应喜好物件，解锁专属剧情对话（参见 [[07 喜好物件系统]]）  
3. **真相层**：主线推进至对应章节后，解锁该角色的隐藏线索对话

**好感度不强制要求**，但全满好感度可解锁：
- 石伯的"脚印"彩蛋
- 冯叙转交芳华信件
- 裴昀展示原信彩蛋
- 福宝"五岁进山庄"彩蛋
- 幽篁夫人离开时的独白

## 日常节奏周期

| 时间段 | 推荐玩家行为 | 游戏反馈 |
|---|---|---|
| 早晨 | 浇花圃、与石伯对话 | 随机"石伯忘事"小对话 |
| 上午 | 合并修缮材料、完成关卡 | 区域修缮进度推进 |
| 下午 | NPC 来访，赠送物件 | 好感度上升，对话解锁 |
| 傍晚 | 探索新解锁区域 | 发现线索/彩蛋 |
| 夜晚 | 院落复盘，幽兰草开花动画 | 情感小节点 |
