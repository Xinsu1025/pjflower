## 核心机制来源：June's Journey

本游戏以 **《June's Journey》** 的点击寻物玩法为核心原型，结合院落修缮与剧情解锁。玩家在各场景中点击找出隐藏物品，赚取星星积分，用积分解锁新区域并进行装修布置，推动剧情章节向前推进。

---

## 机制映射表

| June's Journey 原机制 | 花开寻踪的对应设计 | 目的 |
|---|---|---|
| 点击场景找到隐藏物品 | 在山庄各区域场景中寻找隐藏物件（器具、线索、植物等） | 给玩家探索感，短循环正反馈 |
| 完成场景获得星星 | 完成寻物场景积累"花印"积分 | 积分可视化，驱动玩家复玩 |
| 用星星解锁区域/装饰物 | 用"花印"解锁院落区域，购入装饰物布置山庄 | 空间进度与审美满足感 |
| NPC 特殊任务/剧情触发 | NPC 喜好物件赠送触发专属对话 | 角色情感加深 |
| 章节推进解锁新场景 | 修缮进度解锁剧情章节与新场景 | 看与修合一 |

## 关卡解锁节奏

```
第一幕（立足篇）
  寻物场景：正院·石板路·花圃外围
  → 积累花印：解锁茶室、正院装修
  → 剧情解锁：序章 + 第1-3章

第二幕（麻烦篇）
  寻物场景：茶室·厨房·水榭·阁楼
  → 积累花印：解锁后院矮墙、阁楼藏书室装修
  → 剧情解锁：第4-8章

第三幕（真相篇）
  寻物场景：后山路·后山花圃·山庄全景
  → 积累花印：解锁山腰平台、全景山庄装修
  → 剧情解锁：第9-12章
```

## NPC互动节奏设计

每位 NPC 有三层互动进度：

1. **出场层**：NPC 初次登场，基础对话解锁  
2. **喜好层**：赠送礼物提升好感度——每位 NPC 均有**六阶礼物体系**（一阶·相识礼 → 六阶·至情礼），阶位越高解锁内容越深（参见 [[07 喜好物件系统]]）  
3. **真相层**：主线推进至对应章节后，解锁该角色的隐藏线索对话

**好感度不强制要求**，但全满好感度可解锁：
- 石伯的"脚印"彩蛋
- 冯叙转交芳华信件
- 裴昀展示原信彩蛋
- 福宝"五岁进山庄"彩蛋
- 幽篁夫人离开时的独白

## 日常节奏周期

| 时间段 | 推荐玩家行为 | 游戏反馈 |
|---|---|---|
| 早晨 | 浇花圃、与石伯对话 | 随机"石伯忘事"小对话 |
| 上午 | 在各区域完成寻物场景、积累花印 | 区域修缮/装饰进度推进 |
| 下午 | NPC 来访，赠送物件 | 好感度上升，对话解锁 |
| 傍晚 | 探索新解锁区域 | 发现线索/彩蛋 |
| 夜晚 | 院落复盘，后山花圃开花动画 | 情感小节点 |
