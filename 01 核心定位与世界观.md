## 一句话定位

> **一个爱乱跑的娘、一座花开不败的旧庄子、几个凑在一起的年轻人，和仙湖镇每天鸡毛蒜皮又暖到心里的日子。**

## 调性关键词

**美好（花）/ 治愈 / 田园 / 接地气 / 笑着笑着的感动**

>问题要有，解决问题的过程要有趣；冲突往往源于误会和沟通不畅；最终的解决从不依靠对抗，而是靠人心和一顿饭。

参考基调：温暖市井喜剧——有笑有泪有悬疑，热闹接地气，寻找之旅、建设之旅、成长之旅。

## 世界观设定

**时代背景：** 明初风格的架空世界，江南水乡，商贾文化与江湖势力并存，志怪不只在小说之中。

**核心地点：** **仙湖镇** — 依山傍水的小镇，镇边有座**花隐山**，山中一座快废弃的花隐山庄。

**山庄全名：花隐山庄**

> 坐落于花隐山山腰，因山中四季百花常开、不败不落而得名"花隐"。花家是花隐山世代守山人，主角母亲花芳华毕生研究山中奇异草木，留下大量手记，也因此引来觊觎。

**花隐山的奇景：** 四季同框、百花不灭——此山中春夏秋冬的花卉同时盛开，经年不谢，镇里人习以为常，却说不清为何。传说，奇景的根源，是山中有山神庇佑。

**世界氛围关键词：** 百花常开、山中岁月、日常快乐、探索谜团

**神奇元素占比（约 20%）：** 类《星露谷物语》，奇异之物自然出现在日常生活里——精怪与人同处、山神以孩童为邻、药草有灵——但不喧宾夺主，始终是"温暖日常的背景光"。

---

## 守山人设定

> 花家世代居住于花隐山附近，是传承已久的**守山人**。

| 层次          | 内容                                    |
| ----------- | ------------------------------------- |
| **表层职责**    | 照看山庄、研究山中药草植物                         |
| **真实使命**    | 守护花隐山山神，护住这片山的灵气与奇景不受侵扰               |
| **传承方式**    | 代代口耳相传，在特定时机才会知晓全貌；花如意归来时，尚不知自己的家族身份  |
| **山中百花的意义** | 山中花开不败是山神存在的体现——花家世代研究山中奇异草木，是职责，也是陪伴 |

> 花如意的成长核心之一：从"修庄子的小庄主"到"理解守山人是什么意思"——她娘走之前为何护下福宝，她自己为何在这里，这片山为何对她来说不只是一个家。

## 修仙暗线设定

游戏主线是温暖治愈的日常，但底层埋有一条**隐性修仙线**，不明写，通过以下形式渗透：

| 暗线内容             | 呈现方式                        |
| ---------------- | --------------------------- |
| 幽篁夫人是邪修，已活出寻常人寿数 | 口癖「弹指之间」+行事风格，不作明确注释        |
| 福宝是花隐山山神化身       | 精怪互动、山中奇景、福宝"说漏嘴"的只言片语      |
| 花家守山人传承          | 花芳华手记、石伯偶尔说出的只字片语、终章        |
| 后山花圃的灵性          | 福宝在的地方花开不败，夜里花圃隐有微光（但不作解释）  |
| 霓裳或许也不寻常         | 极少数场景里，霓裳会出现在不该出现的地方，然后装作没事 |

> **设计原则：** 不破坏主基调，不强迫玩家理解暗线。会看的人看见了会惊喜；不看的人享受温暖日常，同样完整。

## "花"主题贯穿全局的设计逻辑

| 层次       | 体现方式                                     |
| -------- | ---------------------------------------- |
| **人物命名** | 主角及家族以花为名，正派配角带草木意象                      |
| **场所命名** | 山庄各院落以花命名，修缮后"花开"解锁新剧情                   |
| **母亲线索** | 母亲失踪前种下的各种花卉隐藏线索，"花开时节"触发关键记忆            |
| **山神联结** | 山庄花卉不败，与山神福宝的驻守密切相关；庄子修好一处，花开一处，暗合山神"醒来" |
| **守山传承** | 山中花开不败是山神灵气的显化，花家世代研究山中草木，是守护，也是陪伴       |
| **玩法机制** | 点击寻物场景积累"花印" → 解锁区域 → 装饰布置 → 触发剧情章节      |
|          |                                          |
