## 反派阵营

### 金不换 — "感觉良好先生"

| 基础信息 | 内容 |
|---------|------|
| **性别** | 男 |
| **年龄** | 22 岁 |
| **职业** | 金玉山庄少庄主，家业靠父亲，本人更擅长斗鸡走马 |
| **服装关键色** | 大红、明黄、描金——颜色永远最响，配饰永远比场合隆重三个等级 |
| **喜好物件** | [[07 喜好物件系统#金不换·描金折扇\|描金折扇（六阶礼物体系）]] |

**性格：**  
永远活在"我已经赢了"的自我叙事里。计划周详、执行一塌糊涂。本质不坏，只是从小被惯坏、被[[#幽篁夫人|幽篁夫人]]拿捏着，蠢过头了显出几分可怜。

**真实动机（金玉山庄）：**  
金不换本人对幽篁夫人的秘密一无所知。他来花隐山庄，是因为觉得这么好的山庄破落下去实在可惜——依山傍水、四季花开，这样的地方若能整修开发，必是江南一绝。他的目的是**商业开发**，不是抢夺，只是手段欠考量、执行一团糟。

> 金不换（看着院子里盛开的花）："这地方，若是整治好了，来往客商都要停脚看一眼。  
> （转头）"花姑娘，你有没有想过，合作？"  
> 花如意（端着茶）："没有。"  
> 金不换："……合伙呢？"  
> 花如意："也没有。"  
> 金不换：**"……"** （折扇合上，耳根微红）"我是真心的。"

**固定喜剧结构：**

```
金不换内心独白："今天必胜，我准备得很充分。"
      ↓
登门，被石伯以最隆重的礼节迎进去，喝了两盏茶
      ↓
福宝跑来问他"你今天又来干嘛"
      ↓
他刚要开口，霓裳飞过来停在他头上
      ↓
花如意出来，平静地说了一句话让他彻底失去主场
      ↓
金不换回去的路上："下次一定。"
```

**三幕演变：**

| 阶段 | 他的自我感觉 | 实际情况 |
|------|------------|---------|
| 第一幕 | "一个小姑娘，随便来几次就拿下了" | 来了五次，每次喝完茶空手回去 |
| 第二幕 | "幽篁说这次的计划万无一失" | 他是幽篁夫人计划里最显眼的棋子，目标是掩护她找人 |
| 第三幕 | "这次是我主动帮花如意，我是关键人物" | 确实帮上了，但方式是"意外说漏嘴" |

**成长弧光：**
```
以为自己在做大事的纨绔少庄主，其实是幽篁夫人的遮掩工具
  → 在花隐山庄屡次败于"热闹"，开始隐隐觉得这里好像比他家那边有意思
    → 福宝问"你其实不太想来抢吧"，他第一次认真想了想
      → 终章：意外说漏嘴帮了花如意，撑着说"本少爷早有此意"，耳根红了
```

### 幽篁夫人 — "神秘的真正反派"

| 基础信息 | 内容 |
|---------|------|
| **性别** | 女 |
| **年龄** | 不详（外貌三十多岁，实际年龄无从考据） |
| **职业** | 表面是金玉山庄总管事，实则是寻访天地灵物的邪修，金玉山庄只是她借用的掩护 |
| **服装关键色** | 深墨绿色外袍，竹纹暗花，腰间一块翡翠；永远以一方薄纱半遮面，露出的眼睛极静，极冷 |
| **喜好物件** | [[07 喜好物件系统#幽篁夫人·后山灵花押花\|后山灵花押花（六阶礼物体系）]]|[[07 喜好物件系统#幽篁夫人·后山灵花押花\|后山灵花押花（六阶礼物体系）]] |

**性格：**  
极度精明，极度克制。表情只有两种：温和的微笑，和稍微比温和微笑冷一度的微笑。她不急——她有的是时间，这一点从她的行事风格里处处透出来，让凡人感到一种说不清楚的寒意。

**口癖：**
- "弹指之间，我有得是耐心。"（轻描淡写，却叫人心里发凉）
- "有趣。"（与花如意的判断截然不同——她的"有趣"里没有温度）
- （沉默，微微侧头，看了对方很久，然后才开口）

**真实身份：邪修**

> 幽篁夫人修的是旁门左道，以天地灵物延续寿命，外貌早已不能作为年龄的参照。她本人对"弹指之间"的说法毫不避讳——她说的是字面意思。

**真实动机与行动时间线：**

| 时间 | 幽篁夫人的行动 | 实际情况 |
|------|-------------|---------|
| 五年前 | 循灵气踪迹找到花隐山，发现千年人参（福宝化身）即将到手 | 花芳华发现后出手阻止，将福宝带离安置于山庄 |
| 五年来 | 反复探查花隐山，发现山中植被花卉并未衰落——说明山神仍在山内 | 福宝以孩童形态藏于山庄，幽篁夫人未察觉 |
| 游戏现在 | 借金玉山庄之名，以"总管事"身份渗入仙湖镇，暗中搜寻山神线索 | 金不换对此一无所知，是无意识的掩护工具 |

**神秘层次（逐步解锁）：**

| 层次 | 内容 | 解锁时机 |
|------|------|---------|
| 表层 | 金玉山庄的精明总管事 | 第一幕 |
| 第二层 | 真实目的与商业开发无关，在找一件"山里的东西" | 第二幕中期 |
| 第三层 | 她认识花芳华，五年前两人在花隐山正面交锋过 | 第二幕末 |
| 底层 | 她是邪修，所求之物是花隐山山神——她察觉花隐山四季不凋的秘密背后有天地灵物 | 终章 |

> 真相详见 [[12 幕后线索彩蛋]]

**与花如意的终章对话：**

> 幽篁夫人（看着福宝在花圃里睡着，周围百花开了一圈）："……这孩子，就是那件东西？"  
> 花如意（站在她前面）："她叫福宝。她不是'东西'。"  
> 幽篁夫人沉默片刻，视线没有移开。  
> "花家人，一代守一代。"  
> "你娘当年也是这么拦着我的。"  
> （停顿）"弹指之间，我有得是耐心。"  
> 花如意："但你等到了什么？"  
> 幽篁夫人看了她很久，什么都没说，最后离开。

**结局：** 不是被打败，而是被困住了——她等了五年，等来的是一个不肯让的孩子和一个更不肯让的庄主。离开前，她最后看了一眼福宝，神情里有什么东西，说不清楚，不像算计。

## 特殊角色

### 霓裳 — 粉色小鸟

| 基础信息 | 内容 |
|---------|------|
| **外形关键色** | 粉色羽毛、浅金色尾羽，体型小巧，静止时优雅，行动时混乱 |
| **名字由来** | 苏凝给取的，说"这么漂亮的颜色，不叫霓裳叫什么" |
| **来历** | 花如意回乡途中在驿站附近发现，羽毛被藤条缠住，帮它解开后它再没走 |
| **喜好物件** | [[07 喜好物件系统#霓裳·新鲜红小米\|新鲜红小米（六阶礼物体系）]] |

**性格：** 有主见、会记仇、喜欢凑热闹，但在花如意难过时会安静地待在她肩上。

**与各角色互动：**

| NPC | 互动状态 |
|-----|---------|
| **[[02 主角设定\|花如意]]** | 不听话但不离开，"互相嫌弃但谁也不走"的陪伴 |
| **[[03 NPC-正面配角#福宝\|福宝]]** | 双向奔赴的好朋友，两个都说对方的话，旁人听不懂 |
| **[[03 NPC-正面配角#周朗\|周朗]]** | 表面互相忽视，工具袋里一直放着它爱吃的谷子 |
| **[[03 NPC-正面配角#苏凝\|苏凝]]** | 在苏凝绣花时安静趴着，比任何时候都乖 |
| **[[03 NPC-正面配角#裴昀\|裴昀]]** | 第一次见面就停在他肩上，此后介于接受和监视之间 |
| **[[#金不换\|金不换]]** | 踩过他的文书，此后金不换想发作，它歪头看他，发作不出来 |
| **[[03 NPC-正面配角#石来喜（石伯）\|石伯]]** | 每天早上接受他的问候然后飞走，风雨无阻 |
