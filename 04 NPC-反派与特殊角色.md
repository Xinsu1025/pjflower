## 反派阵营

### 金不换 — "感觉良好先生"

| 基础信息 | 内容 |
|---------|------|
| **性别** | 男 |
| **年龄** | 22 岁 |
| **职业** | 金玉山庄少庄主，家业靠父亲，本人更擅长斗鸡走马 |
| **服装关键色** | 大红、明黄、描金——颜色永远最响，配饰永远比场合隆重三个等级 |
| **喜好物件** | [[07 喜好物件系统#金不换·描金折扇\|描金折扇]] |

**性格：**  
永远活在"我已经赢了"的自我叙事里。计划周详、执行一塌糊涂。本质不坏，只是从小被惯坏、被[[#幽篁夫人|幽篁夫人]]拿捏着，蠢过头了显出几分可怜。

**固定喜剧结构：**

```
金不换内心独白："今天必胜，我准备得很充分。"
      ↓
登门，被石伯以最隆重的礼节迎进去，喝了两盏茶
      ↓
福宝跑来问他"你今天又来干嘛"
      ↓
他刚要开口，霓裳飞过来停在他头上
      ↓
花如意出来，平静地说了一句话让他彻底失去主场
      ↓
金不换回去的路上："下次一定。"
```

**三幕演变：**

| 阶段 | 他的自我感觉 | 实际情况 |
|------|------------|---------|
| 第一幕 | "一个小姑娘，随便来几次就拿下了" | 来了五次，每次喝完茶空手回去 |
| 第二幕 | "陆霜说这次的计划万无一失" | 他是计划里最显眼的那个棋子 |
| 第三幕 | "这次是我主动帮花朝，我是关键人物" | 确实帮上了，但方式是"意外说漏嘴" |

**成长弧光：**
```
以为自己主导一切的纨绔少庄主，实为棋子
  → 在花隐山庄屡次败于"热闹"，开始隐隐觉得哪里不对
    → 福宝问"你其实不太想来抢吧"，他第一次认真想了想
      → 终章：意外说漏嘴帮了花如意，撑着说"本少爷早有此意"，耳根红了
```

### 幽篁夫人 — "神秘的真正反派"

| 基础信息 | 内容 |
|---------|------|
| **性别** | 女 |
| **年龄** | 不详（看起来三十多岁） |
| **职业** | 表面是金玉山庄总管事，实则来历成谜，金玉山庄只是她借用的平台 |
| **服装关键色** | 深墨绿色外袍，竹纹暗花，腰间一块翡翠；永远以一方薄纱半遮面，露出的眼睛极静，极冷 |
| **喜好物件** | [[07 喜好物件系统#幽篁夫人·一枚独枝幽兰押花\|一枚独枝幽兰押花]]（唯一有效礼物） |

**性格：**  
极度精明，极度克制。表情只有两种：温和的微笑，和稍微比温和微笑冷一度的微笑。

**神秘层次（逐步解锁）：**

| 层次 | 内容 | 解锁时机 |
|------|------|---------|
| 表层 | 金玉山庄的精明管事 | 第一幕 |
| 第二层 | 幽兰草的真正觊觎者，真实身份与金玉山庄无关 | 第二幕中期 |
| 第三层 | 她认识[[02 主角设定#花芳华（缺席的母亲）|花芳华]]，两人之间有一段更早的过节 | 第二幕末 |
| 底层 | 她找的不只是幽兰草，而是芳华手记里的另一件东西 | 终章 |

> 真相详见 [[12 幕后线索彩蛋]]

**真实动机：** 追寻幽兰草中据说藏有的一张"古山图"（与她家族旧案有关），花了二十年监视芳华——结果发现芳华根本不知道"古山图"的事，她只是真的在找花。

> 幽篁夫人（沉默良久）："她……真的只是在找花？"  
> 花如意："我娘这辈子，除了我，就只有花了。"  
> 幽篁夫人（闭眼）："……我查了二十年。"

**结局：** 不是输给花如意的聪明，而是输给了她从未将其纳入计算的东西——人情。离开前隔着院门看了一眼盛开的幽兰草，什么都没说，走了。

## 特殊角色

### 霓裳 — 粉色小鸟

| 基础信息 | 内容 |
|---------|------|
| **外形关键色** | 粉色羽毛、浅金色尾羽，体型小巧，静止时优雅，行动时混乱 |
| **名字由来** | 苏凝给取的，说"这么漂亮的颜色，不叫霓裳叫什么" |
| **来历** | 花如意回乡途中在驿站附近发现，羽毛被藤条缠住，帮它解开后它再没走 |
| **喜好物件** | [[07 喜好物件系统#霓裳·新鲜红小米\|新鲜红小米]] |

**性格：** 有主见、会记仇、喜欢凑热闹，但在花如意难过时会安静地待在她肩上。

**与各角色互动：**

| NPC | 互动状态 |
|-----|---------|
| **[[02 主角设定\|花如意]]** | 不听话但不离开，"互相嫌弃但谁也不走"的陪伴 |
| **[[03 NPC-正面配角#福宝\|福宝]]** | 双向奔赴的好朋友，两个都说对方的话，旁人听不懂 |
| **[[03 NPC-正面配角#周朗\|周朗]]** | 表面互相忽视，工具袋里一直放着它爱吃的谷子 |
| **[[03 NPC-正面配角#苏凝\|苏凝]]** | 在苏凝绣花时安静趴着，比任何时候都乖 |
| **[[03 NPC-正面配角#裴昀\|裴昀]]** | 第一次见面就停在他肩上，此后介于接受和监视之间 |
| **[[#金不换\|金不换]]** | 踩过他的文书，此后金不换想发作，它歪头看他，发作不出来 |
| **[[03 NPC-正面配角#石来喜（石伯）\|石伯]]** | 每天早上接受他的问候然后飞走，风雨无阻 |
